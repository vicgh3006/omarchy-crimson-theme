# Crimson - A Red-themed Omarchy Theme

Crimson is a sleek and minimal terminal theme for Omarchy with bold crimson accents.  
The theme's aesthetic is inspired by a minimalist, pixelated illustration of a lakeside sunset or dusk.

**Status:** New updates coming soon.
